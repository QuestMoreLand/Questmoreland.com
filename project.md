# Questmoreland.com

Personal link/profile page for [www.questmoreland.com](https://www.questmoreland.com).

## Stack

- Vite + React + TypeScript
- Deployed automatically to GitHub Pages via GitHub Actions on push to `master`

## Development

```bash
npm install
npm run dev      # local dev server
npm run build    # production build
npm run preview  # preview production build locally
```

## Updating content

No code changes needed for routine updates:

- **Speedrun status** — edit `src/data/speedrun.json`
- **Social links** — edit `src/data/links.json`
